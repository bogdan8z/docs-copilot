# How to Generate a Custom Instructions File

To create a GitHub Copilot instructions file for your project:

1. Review your existing codebase structure and patterns
2. Use this prompt with Copilot to analyze your project:

   ```text
   chat prompt: generate github custom instructions file for this project, please define the patterns, coding guidelines, frameworks, directory structure based on existing codebase
   ```

3. Copilot will generate guidelines based on your code
4. Save the generated instructions to `.github/copilot-instructions.md` in your repository root
